# DS Properties Management System

A full-stack Property Management System developed to help real estate businesses manage customers, plot transactions, expenses, categories, and financial records through a modern dashboard.

## Features

### Dashboard

* Financial overview
* Customer statistics
* Income & Expense summary
* Current balance
* Recent transactions

### Customer Management

* Add customers
* Edit customer details
* Delete customers
* View customer list

### Transaction Management

* Record income and expenses
* Associate income with customers
* Associate expenses with categories
* Edit transactions
* Delete transactions

### Category Management

* Add categories
* Edit categories
* Delete categories

## Tech Stack

### Frontend

* React
* Vite
* Tailwind CSS
* React Router
* React Icons

### Backend

* Node.js
* Express.js
* PostgreSQL

## Project Structure
